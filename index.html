Architecture
System layers
Four layers from hardware to application, each self-healing and boot-persistent.
L1
Hardware
Pi 5Hailo-8NVMe
Five Raspberry Pi nodes with two Hailo-8 AI accelerators (52 TOPS), 1TB NVMe, thermal management, power optimization via conservative governors.
L2
Network
WireGuardCloudflare
Encrypted WireGuard mesh (10.8.0.x) connecting all nodes through Anastasia hub. 18 Cloudflare tunnels route 48+ domains to the fleet.
L3
Services
DockerOllamaGitea
Docker Swarm orchestration, 16 Ollama models, self-hosted Gitea with 207 repos, PostgreSQL, Pi-hole DNS, MinIO object storage.
L4
Autonomy
Self-Healing
Heartbeat checks every minute, healing scripts every five minutes. Automatic service restart, fleet telemetry, power monitoring via cron.
Fleet
Edge nodes
Five Raspberry Pi nodes forming a self-healing WireGuard mesh across the local network.

Quick start
Clone the scripts repo and deploy to your Pi fleet in three commands.
bash
git clone https://github.com/BlackRoad-OS-Inc/blackroad-operator.git
cd blackroad-operator
npm install

# CLI tools (80+ br-* scripts)
./br-fleet.sh status # SSH into all Pis
./br-ai.sh "explain this" # Stream Ollama prompt
./br doctor # Full system health check
./br deploy --target=all # Multi-cloud deploy
